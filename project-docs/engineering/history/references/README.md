# References

Use this directory for external context that can help AI agents understand the project.

Allowed examples:

- Screenshots.
- Design mockups.
- Architecture diagrams.
- API reference notes.
- Research notes.
- External documentation links.
- Sample payloads.
- Competitor or inspiration references.

Do not store secrets, private tokens, large generated artifacts, or runtime data here.

Keep a short note next to any reference explaining why it matters.

For local external repositories that are not Ayati documentation, use the
root-level `reference-projects/` directory instead.
