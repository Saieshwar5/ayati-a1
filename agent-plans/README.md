# Agent Plans

This directory contains focused, evolving plans that the AI agent may later
implement. It is a sibling of `agent-notes/`.

Use the directories this way:

```text
agent-notes/
    user ideas, discussions, decisions, investigations, and working notes

agent-plans/
    organized problem definitions, architecture, implementation slices,
    invariants, open questions, and test expectations
```

Plans are working direction rather than stable project documentation. Move
confirmed and implemented architecture into `project-docs/` when it becomes
the actual supported system.

## Current Plans
